# GerberParser — Gerber File Viewer with G-Code Generator

Графический просмотрщик Gerber-файлов с генератором G-кода для автоматического диспенсера паяльной пасты.

## Возможности

### Поддержка Gerber-файлов
- Форматы: `.gbr`, `.ger`, `.gtp`, `.gpb`, `.pho`, `.cmp`, `.sol`, `.top`, `.bot`, `.gko`, `.gtl`, `.gbl`, `.gts`, `.gbs`, `.gto`, `.gbo`, `.gpt`
- Парсинг апертур (круги, прямоугольники)
- Парсинг регионов G36/G37
- Парсинг линий и дуг
- Загрузка нескольких файлов одновременно

### Визуализация слоя (главное окно)
- **Полигоны** — голубая заливка с контуром (паста, медные области)
- **Линии** — жёлтые линии (дорожки)
- **Сетка** — адаптивный шаг
- **Зум** — колёсико мыши
- **Панорамирование** — **ЛКМ** + перетаскивание
- Координаты курсора (X, Y) в статусной строке

### Генератор G-кода
- Преобразование выбранного Gerber-слоя в точки дозирования (центры полигонов)
- Расчёт времени дозирования по диаметру
- Сортировка точек по маршруту (Y, затем X)
- Экспорт в G-code для ЧПУ-диспенсера

### Окно G-code (отдельное окно)
- Слева — схема точек и маршрута, справа — текст G-code
- Рулетки X/Y (мм), настраиваемые цвета и шаг сетки (меню **«Вид»**)
- Синхронизация выделения с текстом: клик по точке или по строкам в редакторе подсвечивает соответствующие фрагменты; при **мультивыделении** курсор в тексте не прыгает на каждую точку
- Меню **«Вид» → Рулетка измерения** — два щелчка **ЛКМ** по схеме (с привязкой к точке при наведении): длина и ΔX, ΔY в мм; **ПКМ** или **Esc** — сброс
- Меню **«Вид» → Панель выделенных точек** — таблица №, X/Y, диаметр, Z, время дозы; ширина дока подстраивается при изменении размера окна

## Как работать с программой

### 1. Главное окно — просмотр Gerber

1. **Открыть файл(ы):** меню **File → Open** / **Open Multiple** или соответствующие действия в интерфейсе.
2. В списке **Gerber Layers** выберите слой — он отобразится справа.
3. **Масштаб:** колёсико мыши (курсор — якорь зума).
4. **Сдвиг вида:** зажмите **левую кнопку мыши** на пустом месте и перетащите.
5. **G-code по слою:** нажмите **Generate G-Code** на панели инструментов (или аналог в меню). Откроется окно генератора для **текущего выбранного слоя**.

### 2. Окно «G-Code Generator» — точки и текст

| Действие | Результат |
|----------|-----------|
| **ЛКМ** по точке | Выделить одну точку; в тексте подсвечивается её блок |
| Клик в **тексте G-code** (курсор на строках точки) | Выделить соответствующую точку на схеме |
| **Ctrl + ЛКМ** по точкам | Добавить/убрать точку из выделения |
| **ЛКМ** на пустом месте + **перетаскивание** | **Панорамирование** схемы |
| **ПКМ** на пустом месте + **рамка** | Выделить **все точки**, попавшие в прямоугольник |
| **Ctrl** при завершении рамки (ПКМ) | **Добавить** к текущему выделению точки из рамки |
| Короткий **ЛКМ** на пустом месте (без сдвига) | Снять выделение |
| **Alt + ЛКМ** + перетаскивание с точки | Переместить выделенные точки (после выделения) |
| **Shift + ЛКМ** | Добавить новую точку в месте клика |
| **Delete / Backspace** | Удалить выделенные точки |
| **СКМ** + перетаскивание | Панорамирование (как ЛКМ) |
| Колёсико | Зум |
| Меню **Вид → Цвета** | Фон, сетка, точки, выделение |
| Меню **Вид → Шаг сетки** | Авто или фиксированный шаг (мм) |
| Меню **Вид → Рулетка измерения** | Режим двух кликов для расстояния и приращений (см. выше) |
| Меню **Вид → Панель выделенных точек** | Показать/скрыть таблицу по текущему выделению |
| **Reset View** | Сброс масштаба и смещения вида |
| **Copy G-Code** / **Save .gcode** | Буфер обмена или файл |

Под графиком отображаются **X, Y** (мм) под курсором и подсказка/результат **рулетки измерения** (если режим включён). После правок точек G-code в тексте пересобирается; копирование и сохранение берут актуальный текст.

### 3. Сборка

**Требования:** C++17, Qt 5.15+.

В репозитории есть варианты сборки (CMake / `*.pro` / Visual Studio `qt_win/Gerber_Parser.sln` — в зависимости от вашей среды). Соберите проект в вашей IDE или:

```bash
mkdir build && cd build
cmake ..
cmake --build . --config Release
```

### Пример фрагмента G-code

```text
; ===== G-CODE FOR PASTE DISPENSER =====
G21 ; Units in mm
G90 ; Absolute positioning
G00 Z5.0 ; Raise Z
; Point 1 - diameter: 0.900 mm
G00 X38.375 Y38.575
...
M02 ; End of program
```
